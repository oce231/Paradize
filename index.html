Paradize — Configuration
Connecte ta base de données Supabase pour synchroniser les données avec ta guilde en temps réel.
Va sur supabase.com et crée un compte gratuit + un nouveau projet.
Dans SQL Editor, colle et exécute ce script :
CREATE TABLE app_state ( key text PRIMARY KEY, value jsonb, updated_at timestamptz DEFAULT now() ); ALTER TABLE app_state ENABLE ROW LEVEL SECURITY; CREATE POLICY "open" ON app_state FOR ALL USING (true) WITH CHECK (true);
Dans Project Settings → API, copie l'URL et la clé anon public.
Les données sont partagées avec toute ta guilde via cette base.
